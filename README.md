...
1.    git clone https://github.com/Harshksaw/LMS-Frontend.git

...

```
2.    Move into the Directory
       cd LMS-Frontend 
```

```
3.  Install Dependency ->
    npm i
```

```
4. run the server -> 
    npm run dev
```
